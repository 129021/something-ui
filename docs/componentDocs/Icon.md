
# Icon图标
提供了一套常用的图标集合


## 使用方法
直接通过设置`name`为`iconName`来使用即可。例如：

<s-icon name="edit"></s-icon>
<s-icon name="share"></s-icon>
<s-icon name="delete"></s-icon>
<s-icon name="search"></s-icon>
